WELCOME TO CODE SYNC CONFERENCES

WHAT IS CODE SYNC?

Code Sync is a family of tech conferences. A brand that consolidates all the conferences that we’ve been organizing since 2008 retaining their individual personalities and focuses as:

  • Code BEAM
    Discovering the Future of the Erlang Ecosystem
  • Code BEAM Lite
    Satellite events of Code BEAM
  • Code Elixir LDN
    Connecting the Elixir Community
  • Code Mesh LDN
    Exploring Alternative Tech

OUR COMMITMENT

The creation of the Code Sync family of tech conferences is part of the commitment we have made to open our conferences to a wider audience and to spread the culture of Share. Learn. Inspire. globally. And our long-run goal to enhance the value and overall experience when attending any of the conferences belonging to the Code Sync family.

WHY ATTEND?

Apart from great speakers, inspiring talks, battle stories and plenty of coffee, our conferences bring developers together as a community, to learn; share knowledge and ideas; and to inspire one another. See below which conference is next and don’t just take our word for it, check it for yourself!
