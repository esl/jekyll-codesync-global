WHY ATTEND CODE ELIXIR LDN

Code Elixir LDN is a one-day conference bringing together the Elixir community to share, learn and inspire. We aim to encourage diversity and inclusion at Code ELixir LDN, to do this we offer free scholarship and diversity tickets.

Thanks for attending Code Elixir LDN 18!

Through talks and face-to-face networking, Code Elixir LDN broadened attendees' understanding of Elixir and helped them develop new skills. Code Elixir LDN is a platform for the Elixir community to get involved in the future of Elixir's development, present their ideas, showcase new tools and techniques and unlock business development opportunities.

Watch talks from Code Elixir LDN 18!

Networking at Code Elixir LDN

 

Like last year, there will be lightning talks, where it will be your chance to tell the community what you are working on, send your proposal now.

There will also be break-out sessions during the conference. Have a topic, subject, tool or theme you want to discuss with fellow attendees. Tell us what sessions you would like to promote, nominate a break-out session now.
